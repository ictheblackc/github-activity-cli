
# github-activity-cli

![Preview](preview.gif)

**github-activity-cli** is a small command-line application for fetching and displaying recent GitHub activity for a given user. 


> **Note:** This project was created as a solution for the [GitHub User Activity](https://roadmap.sh/projects/github-user-activity) exercise from [roadmap.sh](https://roadmap.sh) to help beginners practice building a CLI app. Some details were simplified, but the code remains a valid example for reference.

## Installation

```
git clone git@github.com:ictheblackc/github-activity-cli.git
```

## Usage

**The general syntax**
```bash
python github-activity-cli.py [username]
```
